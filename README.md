# DrinkSalesSystem BE

Node.js + Express + MongoDB backend for a drink sales system.

## Setup

```bash
npm install
cp .env.example .env
npm run dev
```

## Features scaffolded

- RESTful API structure
- JWT authentication and role-based authorization
- MongoDB models for core business entities
- Upload handling with Multer
- Socket.IO bootstrap
- MongoDB transaction helper

## API base

- `/api/health`
- `/api/auth`
- `/api/users`
- `/api/roles`
- `/api/categories`
- `/api/products`
- `/api/vouchers`
- `/api/uploads`
- `/api/conversations`
- `/api/messages`
- `/api/orders`
- `/api/reviews`
